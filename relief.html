Relief¶

Para el algoritmo de comparación, se ha implementado el algoritmo greedy Relief. Este algoritmo es el siguiente:

algorithms.relief._relief(X, Y)[fuente]¶

Relief algorithm.

Parámetros

  • X – Train input data

  • Y – Train label data

Como el resto de algoritmos, está envuelto en una clase para su integración con el script principal:

class algorithms.relief.Relief(threshold=0.2, seed=77766814)[fuente]¶

Wrapper class for Relief algorithm that provided sklearn-based syntax.
