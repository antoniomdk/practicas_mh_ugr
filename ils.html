Búsqueda Local Reiterada¶

algorithms.ils.ils(X, y, iters, seed)[fuente]¶

Iterated local search algorithm.

It performs ILS with a limit of 1000 neighbours generated per iteration.

Parámetros

  • X – Input data for fitness evaluation

  • y – Input labels for fitness evaluation

  • iters – The number of iterations. Also the number of application of Local search algorithms.

  • seed – Seed to feed the random number generator.

algorithms.ils.mutate(weights)[fuente]¶

Mutation function for ILS.

It selects randomly 10 percent of the weights and mutates it using Gaussian mutation with sigma = 0.4.

Parámetros

weights – Candidate to apply the mutation

Devuelve

a new candidate

class algorithms.ils.ILS(threshold=0.2, iterations=15, seed=1)[fuente]¶
